# Ladda Studio

**Beşiktaş'ta Afrika Esintisi**

Afrika örgüsü, mısır örgüsü, burgu örgü ve saç uzatma hizmetleri sunan kuaför stüdyosu.

## Hizmetler

- Afrika Örgüsü Uzmanlığı
- Afro Dalgası Saç Modeli
- Mısır Örgüsü Sanatı
- Burgu Örgü
- Örgü Kaynak Hizmeti
- İğne Kaynak Saç Uzatma

## Tech Stack

- Next.js 14+ (App Router, TypeScript)
- Tailwind CSS
- Framer Motion
- Lucide React
- Fontlar: Playfair Display, Inter

## Başlangıç

```bash
npm install
npm run dev
```
