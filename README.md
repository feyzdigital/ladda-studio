# Ladda Studio

**Your Hair, Your Signature.**

Lüks kuaför ve güzellik stüdyosu. Vogue Magazine estetiğinde, minimalist ve şık web sitesi.

## Tech Stack

- **Next.js 14+** (App Router, TypeScript)
- **Tailwind CSS** + clsx/tailwind-merge
- **Framer Motion** (animasyonlar)
- **Lucide React** (ikonlar)
- **Fontlar:** Playfair Display (başlıklar), Inter (metin)

## Proje Yapısı

```
ladda-studio/
├── app/
│   ├── page.tsx           # Anasayfa
│   ├── services/          # Hizmetler (Cut & Style, Color Studio, vb.)
│   ├── team/              # Ekip
│   ├── gallery/           # Lookbook
│   └── contact/           # Randevu & İletişim
├── components/
├── lib/data.ts            # Hizmetler, ekip, galeri verileri
└── ...
```

## Başlangıç

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)
